## Bundles

Un bundle est un fichier ZIP contenant tous vos Gems et documents téléversés. Utilisez-le pour :

- **Sauvegarder votre configuration** avant d'effectuer de grandes modifications
- **Transférer votre configuration** sur une nouvelle machine
- **Partager un point de départ** avec une autre installation Trove

### Exporter

Cliquez sur **Exporter le bundle** pour télécharger un ZIP de tout ce qui se trouve sur ce serveur. Le fichier est enregistré dans le dossier de téléchargements par défaut de votre navigateur.

### Importer

Cliquez sur **Importer le bundle** et choisissez un fichier ZIP exporté depuis Trove. Deux modes sont disponibles :

- **Ajouter** — fusionne le bundle avec vos Gems et documents existants. Sûr à utiliser avec le contenu actuel ; les doublons sont renommés automatiquement.
- **Remplacer** — supprime tous les Gems et documents actuels avant l'importation. Utilisez cette option lors d'une migration vers une configuration propre ou de la restauration d'une sauvegarde.
