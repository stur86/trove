## Bundles

A bundle is a ZIP file that contains all your Gems and uploaded documents. Use it to:

- **Back up your setup** before making large changes
- **Move your configuration** to a new machine
- **Share a starting point** with another Trove installation

### Exporting

Click **Export bundle** to download a ZIP of everything on this server. The file is saved to your browser's default downloads folder.

### Importing

Click **Import bundle** and choose a ZIP file exported from Trove. Two modes are available:

- **Add** — merges the bundle with your existing Gems and documents. Safe to use alongside current content; duplicates are renamed automatically.
- **Replace** — deletes all current Gems and documents before importing. Use this when migrating to a clean setup or restoring a backup.
