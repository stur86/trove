## 编写好的提示模板

模板是您给 AI 的指令。在文本的任何位置使用 `{{ variable_name }}` 来创建用户在运行 Gem 之前填写的字段（变量名请使用英文字母和下划线）。

**示例：**

```
用 {{ language }} 总结以下文本，使用不超过 {{ max_points }} 个要点：

{{ text }}
```

这将创建三个输入字段：*language*（语言）、*max_points*（最大要点数）和 *text*（文本）。

**编写好提示的技巧：**

- **具体明确** — 告诉模型您希望它产生什么内容。
- **说明格式** — 要点列表、短段落、编号步骤、表格……
- **给出示例** — 如果任务较难，展示一个好答案的样子。
- **保持简洁** — 模型在清晰简洁的指令下效果最佳。
- **清晰命名变量** — `{{ patient_name }}` 比 `{{ name }}` 更好（变量名应能描述其内容）。
