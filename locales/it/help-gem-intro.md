## What is a Gem?

A Gem is a task template that makes a specific AI capability available to users — without them needing to write prompts or understand how AI works.

Each Gem has:

- A **name** and **description** shown on the home screen
- A **prompt template** with `{{ variable }}` placeholders that become input fields for the user
- Optional access to **uploaded documents** from the library

When a user opens a Gem, they fill in the input fields and press **Run**. The AI responds once and the task is complete — there is no back-and-forth chat.

**Examples of useful Gems:**

- *Summarise a care plan* — input: the plan text; output: a bullet-point summary
- *Translate a letter* — inputs: the letter and the target language; output: the translated text
- *Answer a question about school policy* — input: the question; documents: the relevant policy folder
