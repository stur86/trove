# Trove

**Trove** es una plataforma de IA local para usuarios no técnicos, diseñada para instituciones con recursos limitados como escuelas, residencias y centros penitenciarios.

Funciona completamente en tu propio hardware — no se necesita conexión a internet durante el uso normal y ningún dato sale de tus instalaciones.

## Qué hace Trove

Trove permite a un administrador definir **Gems**: tareas de IA reutilizables con un propósito fijo (resumir una carta, traducir un párrafo, responder preguntas a partir de una biblioteca de documentos). Los usuarios eligen un gem, rellenan un formulario corto y obtienen un resultado generado por IA — sin escribir prompts, sin historial de chat y sin interacción libre.

Un servidor, muchos dispositivos. Una vez configurado, cualquier dispositivo en tu red local puede abrir Trove en un navegador.
