# Primeros pasos

Trove ya está en funcionamiento en el servidor de tu institución. No necesitas instalar nada. Abre un navegador en cualquier dispositivo conectado a la misma red y ve a la dirección que te dio tu administrador (tiene un aspecto como `http://192.168.x.x:7770`).

## La pantalla de inicio

Cuando abres Trove ves una lista de **Gems** — tarjetas de colores, cada una es una tarea de IA específica. Ejemplos:

- *Resumir una carta* — pega una carta, obtén un resumen con viñetas
- *Traducir al italiano* — escribe o pega texto, elige un idioma, obtén la traducción
- *Responder a partir de documentos* — haz una pregunta; la IA busca en una biblioteca de documentos aprobada para encontrar la respuesta

Elige el gem que coincida con lo que quieres hacer.

## Qué ocurre a continuación

Cada gem muestra un formulario corto. Rellena los campos, adjunta una imagen si la tarea lo pide, y pulsa **Ejecutar**. La IA procesa tu solicitud y muestra el resultado en pantalla.

No hay historial de chat ni memoria entre ejecuciones. Cada ejecución es independiente.
