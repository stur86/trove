# Entrada multimedia

Algunos gems aceptan una imagen además de los campos de texto. Esta página explica cómo funciona.

## Entrada de imagen

Cuando un gem está configurado con soporte de imagen, verás un botón **Subir imagen** o un icono de cámara en el formulario. Puedes:

- **Subir un archivo** — elige una imagen JPEG, PNG o WebP desde tu dispositivo.
- **Hacer una foto** — en dispositivos móviles, el botón de cámara abre directamente la cámara de tu dispositivo.

La imagen se envía a la IA junto con tu texto. Los usos habituales incluyen leer un documento escrito a mano, describir una fotografía o extraer texto de una captura de pantalla.

!!! note "Tamaño de imagen"
    Las imágenes muy grandes se reducen automáticamente antes de enviarse. Mantén las imágenes por debajo de 5 MB para obtener los mejores resultados.

## Entrada de audio

La entrada de audio no está disponible actualmente. Se activará en una futura actualización.

## Gems sin entrada multimedia

Si el gem que estás usando no tiene campo de imagen o audio, esas opciones no aparecerán. El administrador controla qué capacidades tiene cada gem.
