# Per iniziare

Trove è già in esecuzione sul server della tua istituzione. Non è necessario installare nulla. Apri un browser su qualsiasi dispositivo collegato alla stessa rete e vai all'indirizzo che ti ha comunicato il tuo amministratore (ha un aspetto simile a `http://192.168.x.x:7770`).

## La schermata principale

Quando apri Trove vedi un elenco di **Gem** — schede colorate, ognuna delle quali rappresenta un'attività AI specifica. Esempi:

- *Riassumi una lettera* — incolla una lettera, ottieni un riassunto in punti elenco
- *Traduci in italiano* — scrivi o incolla del testo, scegli la lingua, ottieni la traduzione
- *Rispondi dai documenti* — fai una domanda; l'AI cerca nella raccolta di documenti approvata per trovare la risposta

Scegli il Gem adatto a ciò che vuoi fare.

## Cosa succede dopo

Ogni Gem mostra un breve modulo. Compila i campi, allega un'immagine se il task lo richiede, e premi **Esegui**. L'AI elabora la tua richiesta e mostra il risultato sullo schermo.

Non esiste cronologia delle conversazioni né memoria tra un'esecuzione e l'altra. Ogni esecuzione è indipendente.
