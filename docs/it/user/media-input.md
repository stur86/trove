# Input multimediale

Alcuni Gem accettano un'immagine insieme ai campi di testo. Questa pagina spiega come funziona.

## Input immagine

Quando un Gem è configurato con supporto immagine, nel modulo compare un pulsante **Carica immagine** o un'icona fotocamera. Puoi:

- **Caricare un file** — scegli un'immagine JPEG, PNG o WebP dal tuo dispositivo.
- **Scattare una foto** — sui dispositivi mobili, il pulsante fotocamera apre direttamente la fotocamera del dispositivo.

L'immagine viene inviata all'AI insieme al testo. Usi tipici includono la lettura di un documento scritto a mano, la descrizione di una fotografia o l'estrazione di testo da uno screenshot.

!!! note "Dimensione dell'immagine"
    Le immagini molto grandi vengono ridimensionate automaticamente prima dell'invio. Per risultati migliori, usa immagini sotto i 5 MB.

## Input audio

L'input audio non è attualmente disponibile. Sarà abilitato in un aggiornamento futuro.

## Gem senza input multimediale

Se il Gem che stai usando non ha un campo immagine o audio, queste opzioni non compaiono. È l'amministratore a controllare quali funzionalità ha ciascun Gem.
