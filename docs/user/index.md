# Getting started

Trove is already running on your institution's server. You do not need to install anything. Open a browser on any device connected to the same network and go to the address your administrator gave you (it looks like `http://192.168.x.x:7770`).

## The home screen

When you open Trove you see a list of **Gems** — coloured cards, each one a specific AI task. Examples:

- *Summarise a letter* — paste in a letter, get a bullet-point summary
- *Translate to Italian* — type or paste text, choose a language, get the translation
- *Answer from documents* — ask a question; the AI looks through an approved document library to find the answer

Pick the gem that matches what you want to do.

## What happens next

Each gem shows a short form. Fill in the fields, attach an image if the task asks for one, and press **Run**. The AI processes your request and displays the result on screen.

There is no chat history and no memory between runs. Each run is independent.
