# Media input

Some gems accept an image alongside the text fields. This page explains how that works.

## Image input

When a gem is set up with image support, you will see an **Upload image** button or a camera icon on the form. You can:

- **Upload a file** — choose a JPEG, PNG, or WebP image from your device.
- **Take a photo** — on mobile devices, the camera button opens your device's camera directly.

The image is sent to the AI together with your text. Typical uses include reading a handwritten document, describing a photograph, or extracting text from a screenshot.

!!! note "Image size"
    Very large images are automatically scaled down before being sent. Keep images under 5 MB for best results.

## Audio input

Audio input is currently unavailable. It will be enabled in a future update.

## Gems without media input

If the gem you are using does not have an image or audio field, those options will not appear. The administrator controls which capabilities each gem has.
