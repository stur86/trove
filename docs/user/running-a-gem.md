# Running a gem

## Step 1 — Choose a gem

The home screen lists all the gems your administrator has created. Each card shows the gem's name and a short description. Tap or click the one you want.

## Step 2 — Fill in the form

Every gem has its own form. The fields are defined by the administrator and labelled with plain-language names such as *Text to summarise* or *Target language*.

- **Text fields** — type or paste your content directly.
- **Image field** — if the gem accepts images, a file picker or camera button appears. Choose a photo or screenshot from your device.

## Step 3 — Run

Press the **Run** button. The AI begins generating a response and the result appears on screen as it streams in. For short tasks this takes a few seconds; longer documents may take up to a minute.

## Step 4 — Read and copy the result

The result is displayed with formatting — headings, bullet points, bold text, and tables are rendered when the AI uses them. You can select and copy any part of the text as normal. There is no save or history feature — copy what you need before leaving the page.

## Starting over

Press **Run again** (or navigate back to the home screen) to start a fresh run. Nothing from the previous run is carried over.

!!! tip "If the result looks wrong"
    Try rephrasing the content in the input fields. If the problem repeats, let your administrator know — they may need to adjust the gem's instructions.
