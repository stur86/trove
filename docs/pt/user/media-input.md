# Entrada multimédia

Alguns gems aceitam uma imagem além dos campos de texto. Esta página explica como funciona.

## Entrada de imagem

Quando um gem está configurado com suporte de imagem, verá um botão **Carregar imagem** ou um ícone de câmara no formulário. Pode:

- **Carregar um ficheiro** — escolha uma imagem JPEG, PNG ou WebP do seu dispositivo.
- **Tirar uma fotografia** — em dispositivos móveis, o botão de câmara abre diretamente a câmara do seu dispositivo.

A imagem é enviada para a IA juntamente com o seu texto. As utilizações típicas incluem ler um documento manuscrito, descrever uma fotografia ou extrair texto de uma captura de ecrã.

!!! note "Tamanho da imagem"
    As imagens muito grandes são automaticamente reduzidas antes de serem enviadas. Mantenha as imagens abaixo de 5 MB para melhores resultados.

## Entrada de áudio

A entrada de áudio não está disponível atualmente. Será ativada numa atualização futura.

## Gems sem entrada multimédia

Se o gem que está a usar não tiver campo de imagem ou áudio, essas opções não aparecerão. O administrador controla as capacidades de cada gem.
