# Primeiros passos

O Trove já está a funcionar no servidor da sua instituição. Não precisa de instalar nada. Abra um browser em qualquer dispositivo ligado à mesma rede e aceda ao endereço que o seu administrador lhe indicou (tem um aspeto como `http://192.168.x.x:7770`).

## O ecrã inicial

Quando abre o Trove vê uma lista de **Gems** — cartões coloridos, cada um com uma tarefa de IA específica. Exemplos:

- *Resumir uma carta* — cole uma carta, obtenha um resumo em tópicos
- *Traduzir para italiano* — escreva ou cole texto, escolha um idioma, obtenha a tradução
- *Responder a partir de documentos* — faça uma pergunta; a IA pesquisa numa biblioteca de documentos aprovada para encontrar a resposta

Escolha o gem que corresponde ao que pretende fazer.

## O que acontece a seguir

Cada gem apresenta um formulário curto. Preencha os campos, anexe uma imagem se a tarefa o pedir, e prima **Executar**. A IA processa o seu pedido e apresenta o resultado no ecrã.

Não há histórico de conversas nem memória entre execuções. Cada execução é independente.
