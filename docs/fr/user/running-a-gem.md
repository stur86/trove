# Exécuter un gem

## Étape 1 — Choisir un gem

L'écran d'accueil liste tous les gems créés par votre administrateur. Chaque carte affiche le nom du gem et une courte description. Appuyez ou cliquez sur celui que vous souhaitez.

## Étape 2 — Remplir le formulaire

Chaque gem possède son propre formulaire. Les champs sont définis par l'administrateur et étiquetés avec des noms en langage courant tels que *Texte à résumer* ou *Langue cible*.

- **Champs de texte** — tapez ou collez votre contenu directement.
- **Champ image** — si le gem accepte des images, un sélecteur de fichier ou un bouton appareil photo apparaît. Choisissez une photo ou une capture d'écran depuis votre appareil.

## Étape 3 — Lancer

Appuyez sur le bouton **Lancer**. L'IA commence à générer une réponse et le résultat apparaît à l'écran au fur et à mesure. Pour les tâches courtes, cela prend quelques secondes ; les documents plus longs peuvent prendre jusqu'à une minute.

## Étape 4 — Lire et copier le résultat

Le résultat est affiché avec une mise en forme — titres, points, texte en gras et tableaux sont rendus lorsque l'IA les utilise. Vous pouvez sélectionner et copier n'importe quelle partie du texte normalement. Il n'y a pas de fonction d'enregistrement ni d'historique — copiez ce dont vous avez besoin avant de quitter la page.

## Recommencer

Appuyez sur **Relancer** (ou retournez à l'écran d'accueil) pour démarrer une nouvelle exécution. Rien de l'exécution précédente n'est conservé.

!!! tip "Si le résultat semble incorrect"
    Essayez de reformuler le contenu dans les champs de saisie. Si le problème se répète, informez votre administrateur — il devra peut-être ajuster les instructions du gem.
