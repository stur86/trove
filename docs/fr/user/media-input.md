# Saisie multimédia

Certains gems acceptent une image en plus des champs texte. Cette page explique comment cela fonctionne.

## Saisie d'image

Lorsqu'un gem est configuré avec la prise en charge des images, vous verrez un bouton **Téléverser une image** ou une icône d'appareil photo dans le formulaire. Vous pouvez :

- **Téléverser un fichier** — choisissez une image JPEG, PNG ou WebP depuis votre appareil.
- **Prendre une photo** — sur les appareils mobiles, le bouton appareil photo ouvre directement la caméra de votre appareil.

L'image est envoyée à l'IA avec votre texte. Les utilisations typiques incluent la lecture d'un document manuscrit, la description d'une photographie ou l'extraction de texte d'une capture d'écran.

!!! note "Taille de l'image"
    Les images très grandes sont automatiquement réduites avant d'être envoyées. Gardez les images en dessous de 5 Mo pour de meilleurs résultats.

## Saisie audio

La saisie audio n'est pas disponible actuellement. Elle sera activée dans une prochaine mise à jour.

## Gems sans saisie multimédia

Si le gem que vous utilisez ne dispose pas de champ image ou audio, ces options n'apparaîtront pas. L'administrateur contrôle les capacités de chaque gem.
