# Erste Schritte

Trove läuft bereits auf dem Server Ihrer Einrichtung. Sie müssen nichts installieren. Öffnen Sie einen Browser auf einem beliebigen Gerät, das mit demselben Netzwerk verbunden ist, und rufen Sie die Adresse auf, die Ihnen Ihr Administrator mitgeteilt hat (sie sieht aus wie `http://192.168.x.x:7770`).

## Der Startbildschirm

Wenn Sie Trove öffnen, sehen Sie eine Liste von **Gems** — farbige Karten, jede eine bestimmte KI-Aufgabe. Beispiele:

- *Einen Brief zusammenfassen* — fügen Sie einen Brief ein, erhalten Sie eine Zusammenfassung als Aufzählungspunkte
- *Ins Italienische übersetzen* — geben Sie Text ein oder fügen Sie ihn ein, wählen Sie eine Sprache, erhalten Sie die Übersetzung
- *Aus Dokumenten antworten* — stellen Sie eine Frage; die KI durchsucht eine freigegebene Dokumentenbibliothek, um die Antwort zu finden

Wählen Sie das Gem, das zu dem passt, was Sie tun möchten.

## Was als nächstes passiert

Jedes Gem zeigt ein kurzes Formular. Füllen Sie die Felder aus, fügen Sie ein Bild hinzu, wenn die Aufgabe es verlangt, und drücken Sie **Ausführen**. Die KI verarbeitet Ihre Anfrage und zeigt das Ergebnis auf dem Bildschirm an.

Es gibt keinen Gesprächsverlauf und kein Gedächtnis zwischen den Ausführungen. Jede Ausführung ist unabhängig.
