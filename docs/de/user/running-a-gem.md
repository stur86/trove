# Ein Gem ausführen

## Schritt 1 — Ein Gem auswählen

Der Startbildschirm listet alle Gems auf, die Ihr Administrator erstellt hat. Jede Karte zeigt den Namen des Gems und eine kurze Beschreibung. Tippen oder klicken Sie auf das gewünschte Gem.

## Schritt 2 — Das Formular ausfüllen

Jedes Gem hat sein eigenes Formular. Die Felder werden vom Administrator definiert und mit allgemein verständlichen Namen wie *Zusammenzufassender Text* oder *Zielsprache* beschriftet.

- **Textfelder** — geben Sie Ihren Inhalt direkt ein oder fügen Sie ihn ein.
- **Bildfeld** — wenn das Gem Bilder akzeptiert, erscheint eine Dateiauswahl oder eine Kamera-Schaltfläche. Wählen Sie ein Foto oder einen Screenshot von Ihrem Gerät.

## Schritt 3 — Ausführen

Drücken Sie die Schaltfläche **Ausführen**. Die KI beginnt, eine Antwort zu generieren, und das Ergebnis erscheint auf dem Bildschirm, während es eingeht. Für kurze Aufgaben dauert dies einige Sekunden; längere Dokumente können bis zu einer Minute in Anspruch nehmen.

## Schritt 4 — Das Ergebnis lesen und kopieren

Das Ergebnis wird mit Formatierung angezeigt — Überschriften, Aufzählungspunkte, fetter Text und Tabellen werden gerendert, wenn die KI sie verwendet. Sie können jeden Teil des Textes wie gewohnt auswählen und kopieren. Es gibt keine Speicher- oder Verlaufsfunktion — kopieren Sie, was Sie benötigen, bevor Sie die Seite verlassen.

## Von vorne beginnen

Drücken Sie **Erneut ausführen** (oder navigieren Sie zurück zum Startbildschirm), um eine neue Ausführung zu starten. Nichts von der vorherigen Ausführung wird übernommen.

!!! tip "Wenn das Ergebnis falsch aussieht"
    Versuchen Sie, den Inhalt in den Eingabefeldern umzuformulieren. Wenn das Problem weiterhin auftritt, informieren Sie Ihren Administrator — er muss möglicherweise die Anweisungen des Gems anpassen.
