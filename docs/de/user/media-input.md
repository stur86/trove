# Medieneingabe

Einige Gems akzeptieren neben den Textfeldern auch ein Bild. Diese Seite erklärt, wie das funktioniert.

## Bildeingabe

Wenn ein Gem mit Bildunterstützung eingerichtet ist, sehen Sie im Formular eine Schaltfläche **Bild hochladen** oder ein Kamerasymbol. Sie können:

- **Eine Datei hochladen** — wählen Sie ein JPEG-, PNG- oder WebP-Bild von Ihrem Gerät.
- **Ein Foto aufnehmen** — auf mobilen Geräten öffnet die Kamerataste direkt die Kamera Ihres Geräts.

Das Bild wird zusammen mit Ihrem Text an die KI gesendet. Typische Verwendungszwecke umfassen das Lesen eines handgeschriebenen Dokuments, die Beschreibung einer Fotografie oder das Extrahieren von Text aus einem Screenshot.

!!! note "Bildgröße"
    Sehr große Bilder werden vor dem Senden automatisch verkleinert. Halten Sie Bilder für optimale Ergebnisse unter 5 MB.

## Audioeingabe

Audioeingabe ist derzeit nicht verfügbar. Sie wird in einem zukünftigen Update aktiviert.

## Gems ohne Medieneingabe

Wenn das von Ihnen verwendete Gem kein Bild- oder Audiofeld hat, werden diese Optionen nicht angezeigt. Der Administrator legt fest, welche Fähigkeiten jedes Gem hat.
