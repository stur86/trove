# 快速入门

Trove 已在您机构的服务器上运行。您无需安装任何软件。在连接到同一网络的任意设备上打开浏览器，访问管理员告诉您的地址（格式类似 `http://192.168.x.x:7770`）。

## 主页

打开 Trove 后，您会看到一系列 **Gems**——彩色卡片，每张代表一个特定的 AI 任务。示例：

- *摘要信件* — 粘贴一封信，获得要点摘要
- *翻译成意大利语* — 输入或粘贴文本，选择语言，获得翻译
- *从文档回答问题* — 提问；AI 会搜索已授权的文档库来找到答案

选择与您需求相符的 Gem。

## 接下来会发生什么

每个 Gem 会显示一个简短表单。填写字段，如果任务需要请附上图片，然后按**运行**。AI 处理您的请求并在屏幕上显示结果。

没有聊天记录，每次运行之间也没有记忆。每次运行都是独立的。
