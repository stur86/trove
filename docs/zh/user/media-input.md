# 媒体输入

部分 Gems 除文本字段外还接受图片。本页面说明其工作原理。

## 图片输入

当 Gem 配置了图片支持时，表单中会出现**上传图片**按钮或相机图标。您可以：

- **上传文件** — 从设备中选择 JPEG、PNG 或 WebP 图片。
- **拍摄照片** — 在移动设备上，相机按钮会直接打开设备相机。

图片与您的文本一起发送给 AI。典型用途包括读取手写文档、描述照片或从截图中提取文字。

!!! note "图片大小"
    非常大的图片会在发送前自动缩小。将图片保持在 5 MB 以下可获得最佳效果。

## 音频输入

音频输入目前不可用，将在未来的更新中启用。

## 无媒体输入的 Gems

如果您使用的 Gem 没有图片或音频字段，这些选项将不会出现。管理员控制每个 Gem 的功能。
