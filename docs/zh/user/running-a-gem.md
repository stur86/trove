# 运行 Gem

## 第一步 — 选择 Gem

主页列出了管理员创建的所有 Gems。每张卡片显示 Gem 的名称和简短描述。点击您想要的那张。

## 第二步 — 填写表单

每个 Gem 都有自己的表单。字段由管理员定义，并使用通俗易懂的名称标注，例如*要摘要的文本*或*目标语言*。

- **文本字段** — 直接输入或粘贴您的内容。
- **图片字段** — 如果 Gem 支持图片，将出现文件选择器或相机按钮。从设备中选择照片或截图。

## 第三步 — 运行

按**运行**按钮。AI 开始生成回复，结果会实时显示在屏幕上。简短任务通常需要几秒钟；较长的文档可能需要最多一分钟。

## 第四步 — 阅读并复制结果

结果以格式化方式显示——当 AI 使用标题、要点、粗体文本和表格时，这些格式都会正确渲染。您可以像平常一样选择并复制文本的任意部分。没有保存或历史功能——在离开页面之前请复制您需要的内容。

## 重新开始

按**再次运行**（或返回主页）开始新的运行。上次运行的内容不会被保留。

!!! tip "如果结果看起来有误"
    尝试重新表述输入字段中的内容。如果问题反复出现，请通知管理员——他们可能需要调整 Gem 的指令。
