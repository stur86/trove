# 管理员概览

管理面板只能从运行 Trove 的机器上访问。在该机器的浏览器中打开 `http://localhost:7770/admin`，并使用安装时设置的凭据登录。

!!! warning "管理员访问仅限 localhost"
    管理员登录对网络上的所有其他设备都是隐藏的，这是一项安全措施。要管理 Trove，您必须在服务器旁边操作，或使用 SSH 隧道。

## 四个选项卡

| 选项卡 | 您可以执行的操作 |
|---|---|
| **设置** | 选择 AI 模型、设置上下文窗口大小、更改显示语言 |
| **文档** | 上传文件、整理到文件夹中、查看 AI 生成的摘要 |
| **Gems** | 创建、编辑和删除 Gems |
| **日志** | 查看服务器日志的最近 1,000 行，每 5 秒自动刷新 |

## 局域网 URL

设置选项卡显示**局域网 URL**——其他设备访问 Trove 应使用的地址。复制并分享给您的用户。
