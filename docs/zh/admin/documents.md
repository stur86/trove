# 管理文档

文档库允许您让 AI 访问机构的文件——政策、手册、参考表——而无需将它们嵌入到各个 Gem 的 prompt 中。

## 上传文档

1. 在管理面板中打开**文档**选项卡。
2. 选择目标文件夹（或创建新文件夹）。
3. 点击**上传**并选择文件。

支持的格式包括 PDF、Word（`.docx`）、纯文本以及大多数常见办公格式。Trove 使用 [Markitdown](https://github.com/microsoft/markitdown) 在内部将上传的文件转换为纯文本。原始文件与转换版本一起保存。

上传后，AI 会自动生成文档的单行描述。该描述显示在管理面板中，并在 AI 决定查阅哪些文档时使用。

## 文件夹

文档按文件夹组织。文件夹是访问控制的单位：创建 Gem 时，您授予对整个文件夹或其中单个文档的访问权限。

要创建文件夹，在**新建文件夹**字段中输入名称，然后按 Enter（或添加按钮）。

要重命名文件夹或文档，在管理面板中点击其名称。

## AI 如何使用文档

当 Gem 具有文档访问权限时，Trove 会在 AI 开始之前提供所有可访问文档的摘要。AI 然后可以请求其认为相关的任何文档的完整文本。没有向量搜索——AI 从摘要中推理，并按需获取完整内容。

这意味着：
- **简短、命名清晰且描述良好的文档**更容易被 AI 找到和使用。
- **非常大的文档**可能会被截断以适应模型的上下文窗口。
- AI 并不总是使用文档——只有在文档看起来与用户请求相关时才会使用。

## 下载文档

您可以直接从文档选项卡下载单个文档或整个文件夹。

- **文件夹** — 点击文件夹名称旁边的下载图标（↓），接收包含该文件夹中每个文档转换后 Markdown 版本的 ZIP 压缩包。
- **文档** — 点击文档名称旁边的下载图标，接收其转换后的 Markdown 文件（`.md`）。

这些下载包含 Trove 所见的每个文件的纯文本版本，而非原始上传文件。

## 删除文档

点击管理面板中文档旁边的**删除**按钮。文件及其元数据将被永久删除。
